# データ形式について 0.2.0

Ardeckの通信は、最小限のデータ送信ですむように、数バイトに様々なデータをまとめています。

また、実際に送信されるデータは1バイトのチェックサムを付与したうえで[COBS](https://en.wikipedia.org/wiki/Consistent_Overhead_Byte_Stuffing)に従ってエンコードされています。

## デジタルスイッチ

デジタルスイッチのペイロードは `1バイト` です。

| 1bit(MSB) | 6bit | 1bit(LSB) |
| --- | --- | --- |
| `0` | Pin number | `HIGH` or `LOW` |

`HIGH` or `LOW` の実数値は `config.h` の設定によって決定します。
次のコードをご覧ください：

```h
extern const int d_switch_pin[][2] =
  {
    {2, 0}, // 👀
    {3, 0}
  };
```

`{2, 0},` の `0` に注目してください。この部分で、 `LOW` (スイッチが押されていないとき) の時に送信される値を設定できます。  
詳細は、 `README.md` をご覧ください。

## アナログスイッチ

アナログスイッチのペイロードは `2バイト` です。

| 1bit(MSB) | 5bit | 10bit(LSB) |
| --- | --- | --- |
| `1` | Pin number | `0` to `1023` |
